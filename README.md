# 🎨 Tailwind CSS Exercises
## 📖 About This Project

This project contains 20 Tailwind CSS exercises created to practice both fundamental and advanced Tailwind concepts.

Some exercises are standalone components designed to focus on a specific utility or layout technique, while others use content from my existing recipe website to create more realistic examples.

In addition to the required exercises, I also created an extra Shopping Cart page to further practice responsive layouts and component styling.

## 🛠 Key Tailwind Concepts Practiced

Throughout these exercises I worked with:

- Responsive layouts using Flexbox and Grid
- Mobile-first design with responsive breakpoints
- Hover, focus and validation states
- Sticky navigation and sidebar layouts
- Dark mode styling
- Group and peer modifiers
- Shadows, borders and spacing utilities
- Dashboard and card-based UI design

---

## ⭐ Extra Challenge

Besides the required exercises, I created a separate Shopping Cart page.

This allowed me to combine multiple Tailwind concepts into a small realistic feature, including:

- Responsive layouts
- Product cards
- Order summaries
- Navigation between pages
- Consistent visual styling

---

## 📚 What I Learned

This workshop helped me become more comfortable with Tailwind CSS and its utility-first approach. I learned how to build responsive interfaces, create reusable layouts, and add interactive behavior without writing custom CSS.
